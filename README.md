# Tao_Energy_Pill_Adventure

一、設計目的
讓小朋友了解每天好好吃飯睡覺的重要性、花時間學習的價值，以及不是不能休息，但休息太多可能會有反效果。透過每天的選擇持續累積，最終體會「每個人都會變得不一樣」的道理。

二、目標對象與交付形式
對象：國小一年級～六年級
交付形式：單一 HTML 檔案（內嵌 CSS + JS），離線可用，方便分享給老師、家長
裝置適配：需支援手機/平板觸控操作

三、能量丸

元氣充電寶：每天一定要，吃不夠會危害健康

能力大補丸：能讓自己能力值提升，但會消耗電力

神奇泡泡糖：能快速補充電力，但是吃多了效果會降低，甚至影響能力值
